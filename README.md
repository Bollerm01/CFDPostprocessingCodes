# CFDPostprocessingCodes
Postprocessing codes for various CFD platforms for M.S. work

## Folders
- VULCAN codes: Used for processing of the various radii cases for shear layer analysis. Inputs in .plt files
- Volcano ScaLES codes: Used for grid independence, validation, and radii processing. Inputs in .volcano files`
- Archive: Various codes for processing (AVIATION abstract codes and figure generators)
